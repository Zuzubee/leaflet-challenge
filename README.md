Link: https://zuzubee.github.io/leaflet-challenge/

  
## File Organization and Structure

* Located in the parent directory are three folders and an [index.html](https://github.com/Zuzubee/leaflet-challenge/blob/main/index.html) file.
* The [Leaflet-Part-1](https://github.com/Zuzubee/leaflet-challenge/tree/main/Leaflet-Part-1) folder contains a [js](https://github.com/Zuzubee/leaflet-challenge/tree/main/Leaflet-Part-1/js) folder that hasthe [logic.js] (https://github.com/Zuzubee/leaflet-challenge/blob/main/Leaflet-Part-1/js/logic.js) file for part one of the challenge.

* The [Leaflet-Part-2](https://github.com/Zuzubee/leaflet-challenge/tree/main/Leaflet-Part-2) folder contains a [js](https://github.com/Zuzubee/leaflet-challenge/tree/main/Leaflet-Part-2/js) folder that houses the [logic2.js](https://github.com/Zuzubee/leaflet-challenge/blob/main/Leaflet-Part-2/js/logic2.js) file for part two of the challenge.

* The [css](https://github.com/Zuzubee/leaflet-challenge/tree/main/css) folder contains the [style.css](https://github.com/Zuzubee/leaflet-challenge/blob/main/css/style.css) file for styling of the map
